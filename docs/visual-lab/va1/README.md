# VA1 art-direction gate

Status: executable local concept reference; not a runtime route, frozen Phase 0
stimulus, generated production asset, renderer decision, or comprehension result.

Open `index.html?view=compare` from the filesystem to compare three surfaces over
the same information skeleton. Open
`index.html?view=focus&candidate=field-manual` for the current recommended
hypothesis at the fixed 1600×1000 review viewport.

The board intentionally keeps eight room slots, seven worker-action silhouettes,
four independent health signals, the same selected object, the same relation
types, and the same historical cursor in every direction. Only surface language
changes. All labels are repository-authored DOM text; no generated bitmap,
remote dependency, live run, product server route, or production renderer
references this directory.

Cursor truth is anchored to repository fixture
`tests/fixtures/visual_rich_v1.json` at source sequence 30. The selected
Researcher is the event agent, while the associated canonical event subject is
the distinct tool call `tool.logs-1`; the board exposes both fields. Cursor
rooms never use the later context-budget or compaction state. A future seq 37
marker remains on the historical timeline only and is explicitly labelled
`FUTURE`; the label carries `data-for-event-seq="37"` and shares the marker's
position so widening the timeline cannot detach the explanation from the event.

The S1 comparison contract derives one semantic snapshot from that fixture and
requires all three directions to match it across health, eight room states,
evidence data/text, and timeline relations. Visual differences therefore cannot
quietly change the represented run facts.

Promotion requires independent silhouette/readability review, grayscale and
forced-colors checks, zero answer-label clipping, durable hosted screenshots,
and the keep/revise/kill decision recorded in the stage log. Attractive imagery
alone is not an exit gate.
